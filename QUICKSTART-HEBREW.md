# 🚀 מדריך התחלה מהירה

## התחל תוך 5 דקות!

### 1️⃣ הגדרה ראשונית (פעם אחת בלבד)

**משתמשי WINDOWS (הכי קל):**
1. לחיצה כפולה על `setup-windows.bat`
2. המתן עד שההתקנה תושלם
3. זהו! דלג לשלב 2.

**משתמשי MAC (הכי קל):**
1. פתח Terminal
2. גלול לתיקיית הפרויקט
3. הקלד: `bash setup.sh`
4. המתן עד שההתקנה תושלם

**אופציה ידנית:**
```bash
# צור תיקיית public
mkdir public

# העבר קובץ HTML (Windows)
move appointment-reminder-system-hebrew.html public\index.html

# העבר קובץ HTML (Mac/Linux)
mv appointment-reminder-system-hebrew.html public/index.html

# התקן תלויות
npm install
```

### 2️⃣ הפעל את השרת

**WINDOWS:** לחיצה כפולה על `start-server.bat`

**MAC:**
```bash
npm start
```

אמור להופיע:
```
Server running on port 3000
Frontend: http://localhost:3000
Ready to connect to WhatsApp!
```

### 3️⃣ פתח בדפדפן

עבור ל: **http://localhost:3000**

### 4️⃣ התחבר ל-WhatsApp

1. לחץ על כפתור **"התחבר ל-WhatsApp"**
2. יופיע קוד QR
3. פתח WhatsApp בטלפון שלך:
   - **iPhone**: הגדרות → מכשירים מקושרים → קשר מכשיר
   - **Android**: תפריט (⋮) → מכשירים מקושרים → קשר מכשיר
4. סרוק את קוד ה-QR
5. המתן ל-"מחובר ל-WhatsApp" ✅

## ⚡ בעיות נפוצות

### "npm: command not found"
- **פתרון**: התקן Node.js מ-https://nodejs.org/

### קוד QR לא מופיע
- **פתרון**: 
  1. עצור את השרת (Ctrl+C)
  2. הרץ `npm start` שוב
  3. רענן את הדפדפן

### "WhatsApp לא מחובר"
- **פתרון**: לחץ על "התחבר ל-WhatsApp" וסרוק את קוד ה-QR

### הודעות לא נשלחות
- **בדוק**: 
  - WhatsApp מחובר (נורית ירוקה)
  - פורמט מספר הטלפון נכון
  - למקבל יש WhatsApp מותקן

# מרפאה וטרינרית הוליסטית - מערכת תזכורות WhatsApp

מערכת יפה ואוטומטית לשליחת תזכורות WhatsApp לבעלי חיות לפני התורים שלהם.

## 🐾 תכונות

- **אינטגרציה עם WhatsApp**: חבר את חשבון ה-WhatsApp שלך לשליחת תזכורות אוטומטיות
- **תזמון חכם**: קבע תזכורות ליום-3 ימים לפני התורים
- **ממשק יפה**: עיצוב נושא וטרינרי עם צבעים מרגיעים ומקצועיים
- **ניהול לקוחות**: אחסן מידע על בעלי החיות ופרטי התורים
- **תזכורות אוטומטיות**: המערכת שולחת תזכורות באופן אוטומטי בזמנים מתוזמנים
- **שליטה ידנית**: שלח תזכורות מיד במידת הצורך
- **לוח מחוונים סטטיסטי**: עקוב אחר תורים וסטטוס תזכורות

## 📋 דרישות מקדימות

לפני שמתחילים, ודא שיש לך:
- Node.js (גרסה 14 ומעלה)
- npm (מגיע עם Node.js)
- סמארטפון עם WhatsApp מותקן
- Google Chrome או Chromium (נדרש לאוטומציה של WhatsApp Web)

## 🚀 התקנה

### שלב 1: התקן תלויות

פתח את ה-Terminal ונווט לתיקיית הפרויקט, ואז הרץ:

```bash
npm install
```

זה יתקין:
- `express` - מסגרת שרת אינטרנט
- `whatsapp-web.js` - אוטומציה של WhatsApp Web
- `qrcode` - יצירת קוד QR להתחברות WhatsApp
- `node-schedule` - תזמון משימות אוטומטיות
- `cors` - שיתוף משאבים בין מקורות

### שלב 2: הגדר את מבנה הפרויקט

צור תיקיית `public` והעבר את קובץ ה-HTML:

```bash
mkdir public
mv appointment-reminder-system-hebrew.html public/index.html
```

### שלב 3: הפעל את השרת

```bash
npm start
```

השרת יתחיל ב-`http://localhost:3000`

## 📱 איך להשתמש

### התחברות ל-WhatsApp

1. פתח את הדפדפן וגלוש ל-`http://localhost:3000`
2. לחץ על כפתור "התחבר ל-WhatsApp"
3. קוד QR יופיע על המסך
4. פתח WhatsApp בטלפון שלך:
   - **iPhone**: הגדרות → מכשירים מקושרים → קשר מכשיר
   - **Android**: תפריט (⋮) → מכשירים מקושרים → קשר מכשיר
5. סרוק את קוד ה-QR עם הטלפון שלך
6. לאחר החיבור, מחוון הסטטוס יהפוך לירוק

### הוספת תורים

1. מלא את טופס התור עם:
   - שם בעל החיה
   - שם החיה
   - סוג החיה
   - מספר טלפון (עם קוד מדינה)
   - תאריך ושעת התור
   - סוג הטיפול
   - מתי לשלוח תזכורת (1-3 ימים לפני)

2. לחץ על "הוסף תור"

3. התור יופיע בקטע "תורים קרובים"

### ניהול תזכורות

**תזכורות אוטומטיות:**
- המערכת בודקת כל שעה תורים שצריכים תזכורות
- תזכורות נשלחות אוטומטית בשעה 10:00 בבוקר ביום המתוזמן
- דוגמה: אם תור הוא ביום שישי ומוגדר לתזכורת יומיים לפני, התזכורת תישלח ביום רביעי ב-10:00

**תזכורות ידניות:**
- לחץ על "שלח תזכורת עכשיו" בכל תור כדי לשלוח מיד
- שימושי לבדיקה או תזכורות דקה אחרונה

## 🔄 שדרוג לייצור

אב-טיפוס זה משתמש ב:
- אחסון בזיכרון (תורים הולכים לאיבוד בהפעלה מחדש)
- מערכת קבצים מקומית עבור סשן WhatsApp
- ללא אימות משתמשים

לייצור, שקול:

1. **מסד נתונים**: השתמש ב-MongoDB, PostgreSQL, או MySQL
2. **אחסון מתמשך**: אחסן תורים במסד נתונים
3. **אימות משתמשים**: הוסף מערכת התחברות למשתמשים מרובים
4. **אירוח ענן**: פרוס ל-Heroku, DigitalOcean, או AWS
5. **WhatsApp Business API**: שדרג ל-API הרשמי לאמינות טובה יותר

## 🐛 פתרון בעיות

### שגיאת "WhatsApp לא מחובר"

**פתרון**: ודא שלחצת על "התחבר ל-WhatsApp" וסרקת את קוד ה-QR

### קוד QR לא מופיע

**פתרון**: 
1. הפעל מחדש את השרת (`npm start`)
2. נקה את זיכרון המטמון של הדפדפן
3. נסה דפדפן אחר

### הודעות לא נשלחות

**סיבות אפשריות**:
1. סשן WhatsApp Web פג תוקף - התחבר מחדש על ידי סריקת קוד QR
2. פורמט מספר טלפון שגוי - ודא שקוד המדינה כלול
3. WhatsApp לא מותקן בטלפון של המקבל
4. הגבלת קצב מ-WhatsApp

### תורים נעלמים

**סיבה**: אב-טיפוס זה משתמש באחסון בזיכרון

**פתרון**: שדרג לאחסון במסד נתונים (ראה קטע ייצור למעלה)
